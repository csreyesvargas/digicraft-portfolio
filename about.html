About Me

Designer by passion, developer by choice.

I'm a Web and UX Design student at Wake Technical Community College with a real love for front-end development. I like that sweet spot where design meets code and everything just clicks together.

Right now I'm working as a Web Developer and Social Media Manager at Maynor Group LLC, where I'm rebuilding their website from scratch using HTML, CSS, JavaScript, and Astro. Before that I coached at Zebra Robotics, which honestly taught me a lot about how to explain technical stuff in a way that makes sense to people.

I believe good design is invisible. When it works, people don't notice it. They just get what they came for and leave happy. That's what I'm going for.

How I Work

My Process

Every project is a little different, but I always follow the same general path.

01

Discover

Get to know the users, the problem, and the context before touching anything.

02

Define

Map out user flows, organize the information, and plan how it should all work.

03

Design

Build wireframes and high-fidelity screens in Figma until it feels right.

04

Deliver

Test it, refine it, document it. Build it out if the project calls for it.
